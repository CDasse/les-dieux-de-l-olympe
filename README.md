# Les Dieux de l’Olympe
Un projet personnel pour **mettre en pratique mes compétences en HTML et CSS**.

## Présentation

Ce projet est une réalisation personnelle qui m’a permis d’appliquer et de consolider mes compétences en **HTML5**, **CSS3** et un peu de **JavaScript** à travers la création d’un site web statique dédié à la **mythologie grecque**.
Chaque page a été pensée pour illustrer un aspect particulier du développement front-end : **structure, mise en page, animations, et interactions simples**.

## Objectif du projet

L’objectif principal était de montrer ma progression dans l’apprentissage des langages du web en créant un site complet, cohérent et agréable à parcourir.

**Ce projet m’a permis d’apprendre à :**

* organiser le code et les fichiers d’un site web,
* concevoir une interface esthétique et fonctionnelle,
* travailler la sémantique HTML et l’accessibilité,
* maîtriser la mise en forme avec CSS (flexbox, grid, animations),
* gérer la cohérence visuelle grâce à une palette de couleurs et des styles réutilisables.
 
## Compétences techniques développées

### HTML5

Structure sémantique claire (balises header, nav, section, article, footer, etc.).

Création de pages interconnectées (index.html, dieux.html, heracles.html, quiz.html).

Utilisation de liens, listes, images et textes formatés pour une navigation fluide.

Mise en place d’un quiz interactif en intégrant du JavaScript.

### CSS3

Création d’une feuille de style personnalisée et structurée.

Utilisation de Flexbox et CSS Grid pour les mises en page adaptatives.

Gestion des couleurs et typographies avec cohérence visuelle.

Mise en place d’effets de hover et d’animations pour dynamiser l’expérience utilisateur.

Découverte et utilisation d’une palette de couleurs harmonisée (documentée dans palette couleur.pdf).

### JavaScript (bases)

Création d’un petit quiz interactif pour renforcer la dimension ludique du site.

Manipulation du DOM et gestion d’événements simples.

## Ce que j’ai appris

* Comment **structurer** un projet web complet de A à Z.

* L’importance d’un **code clair et bien organisé**.

* Les bonnes pratiques de **mise en page moderne** avec CSS.

* La nécessité de penser à l’**expérience utilisateur** dès la conception.

## Aperçu du site

Le site comporte plusieurs pages :

* **Accueil** (index.html) : introduction au thème

* **Les dieux** (dieux.html) : présentation des divinités de l’Olympe

* **Héraclès** (heracles.html) : focus sur le héros mythologique

* **Quiz** (quiz.html) : test de connaissances interactif

* **Palette de couleurs** (palette couleur.pdf) : document de référence graphique

## Lien et utilisation

Clonez le projet :

```
git clone https://github.com/CDasse/les-dieux-de-l-olympe.git
```
Ouvrez simplement index.html dans votre navigateur pour découvrir le site.
